# Plataforma de Apoio em Saúde Mental para Jovens

Este projeto tem como objetivo criar um ambiente digital que ofereça suporte psicológico, conteúdos educativos sobre saúde mental e canais de contato com profissionais para jovens entre 15 e 30 anos.

## Objetivos
- Disponibilizar informações confiáveis sobre saúde mental
- Facilitar acesso a apoio psicológico
- Reduzir o estigma relacionado à saúde mental
- Criar uma comunidade segura e acolhedora

## Tecnologias utilizadas
- HTML, CSS, JavaScript
- Backend: Node.js
- Banco de dados: MongoDB ou Firebase
